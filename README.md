==========
SortIt-app
==========

Rate things by draging and droping them!

Requirements
------------
* Python (2.7)
* Redis (2.6.0+ because SortIt uses Lua scripts. Notice that Ubuntu's packet manager ships with older)
* Flask
* Flask-And-Redis
* Flask-Testing
* Flask-Script
* Nose
* Scrapy
* Scrapy-Redis

Run
===

```
$ python manage.py run
```

Direct your browser to: localhost:5000
